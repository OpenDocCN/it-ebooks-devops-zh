# Ansible 介绍

## 什么是 Ansible?

Ansilbe 是一个部署一群远程主机的工具。远程的主机可以是本地或者远程的虚拟机，也可以是远程的物理机。

## Ansible 能做什么？

Ansilbe 通过 SSH 协议进行管理节点和远程节点之间的通信。理论上说管理员通过 ssh 到一台远程主机上能做的操作 Ansible 都可以做。

包括：

*   拷贝文件
*   安装包
*   起服务
*   ...
